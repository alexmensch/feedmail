## Feature: Admin Console Enhancements

Adds server-side pagination to the subscriber list, site configuration editing, and credential management to the admin console Settings page. These enhancements build on the core admin console UI and require both API-side changes (pagination parameters) and frontend additions. Credential management gives the admin control over their login email, admin API key, and Resend API key without CLI access.

## Out of scope

- **Customizable page size** — fixed page size; user-configurable sizing is a future enhancement
- **Client-side sorting or searching within subscriber list** — server-side filtering via existing channel/status params only
- **Rate limit config per-worker** — config applies to the feedmail API Worker only; admin Worker rate limits use the same config
- **Resend API key validation during save** — the key is not verified against the Resend API; invalid keys produce errors at send time
- **Admin API key display** — the current key is never shown; only regeneration is available
- **Audit log for credential changes** — no history of when keys were regenerated or email was changed; potential future enhancement
